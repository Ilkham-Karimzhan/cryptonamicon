# cryptonamicon

![Vue.js](https://img.shields.io/badge/vuejs-%2335495e.svg?style=for-the-badge&logo=vuedotjs&logoColor=%234FC08D)

Implement Vue crypto tokens exchange rate system.
- +1000 Tokens rates.
- Update every 3 seconds.
- Tokens rate graphic.
- Use USD to count rate.

Technologies:
- Vue.js
- Option API
- REST API
- LocalStorage
- URL data
